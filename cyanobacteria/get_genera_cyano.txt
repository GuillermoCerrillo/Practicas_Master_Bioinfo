This scripts gathers all the cianobacteria genera form the LPSN database


Input: an url, oin this case "url=https://lpsn.dsmz.de/phylum/cyanobacteriota"

Processing:
	Nested loops, the script crawls through all children taxa of cianobacteria.
	When reaching a genus, it stores it in the output file (genera_cyano.txt)
